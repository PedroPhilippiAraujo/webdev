    Projeto Tutor de IA PT-BR

    O projeto consiste em um programa que leia textos em português e faça correções (gramaticais, ortográficas e sintaxe) utilizando IA. Além disso o usuário pode utilizar outras vantagens como aprimorar o texto com intuito dele ser mais formal e profissional ou resumi-lo em um número de linhas.

    O programa tem dois tipos de usuários, o comum, que utiliza o site normalmente; e o super usuário, que tem acesso aos dados dos usuários comuns. Nesse sentido a primeira aba do site é o cadastro, para cria-lo o usuário precisa digitar seus dados (nome, ...), senha e o tipo de usuário. Dentro do site o usuário comum pode salvar seus textos e os compartilhar entre outros usuários, dentro disso o super usuário consegue visualizar os compartilhamentos feitos dentro do site.

    Recursos usados para o Projeto

    O recurso que utilizaremos no projeto é a API Clound Translation, pois mesmo sendo usada para tradução de textos ela atende os requisitos do programa. Pesquisando sobre APIs de Inteligência Artificial achei vários tipos, como interpretação de texto a fim de análise, conversação e resumo de textos. É provável acrescentar essas e outras IAs ao projeto, mas só o Clound Translation faz o trabalho de verificar a gramática e ortografia de palavras.
